# Hospital Management System (C++ + PostgreSQL)

A console-based Hospital Management System built using C++ and PostgreSQL.

## Features

- Patient Management
- Doctor Management
- Appointment Booking
- PostgreSQL Database Integration

## Technologies

- C++
- PostgreSQL
- libpqxx
- Git

## Build

g++ main.cpp database.cpp patient.cpp doctor.cpp appointment.cpp -lpqxx -lpq -o hospital

## Run

./hospital
